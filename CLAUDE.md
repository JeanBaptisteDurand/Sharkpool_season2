# Sharkpool Season 2

## Project Structure

This repository contains exercises from the Sharkpool Season 2 program. Each exercise lives in its own directory at the repo root.

```
Sharkpool_season2/
├── CLAUDE.md
├── README.md          # Root README with index of all exercises
├── ex01/
│   └── README.md      # Exercise-specific README
├── ex02/
│   └── README.md
└── ...
```

## Conventions

- **One directory per exercise**: each exercise gets its own folder (e.g., `ex01/`, `ex02/`, or a descriptive name).
- **Each exercise has a README.md**: it should contain the exercise title, description/subject, and any instructions or notes.
- **Root README.md**: must be kept up to date with links to every exercise directory. When adding a new exercise, always add it to the root README's exercise list.
- **No language restriction**: exercises may use any language as required by the subject.

## Workflow

1. Create a new directory for the exercise.
2. Add a `README.md` inside it with the exercise subject/description.
3. Add a link to the new exercise in the root `README.md`.
4. Implement the solution in the exercise directory.
