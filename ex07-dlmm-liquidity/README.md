# Exercise 07 - Add DLMM Liquidity on Meteora

## Task

Add DLMM liquidity on Meteora and submit the TX link.

## Instructions

This must be done through the Meteora web interface on **mainnet**.

### Steps

1. **Go to** [meteora.ag](https://app.meteora.ag/)
2. **Connect your wallet** (Phantom, Solflare, etc.) — make sure you have some SOL + tokens
3. **Find a DLMM pool** — pick any pool (e.g. SOL/USDC)
4. **Click "Create Position"** on the right panel
5. **Enter deposit amount** — Auto-Fill will balance both tokens
6. **Choose a volatility strategy:**
   - **Spot** — uniform distribution, good default
   - **Curve** — concentrated around current price, best for stable pairs
   - **Bid-Ask** — more at edges, good for volatile pairs
7. **Set price range** — default is 69 bins centered on active price
8. **Review cost details** and click **"Add Liquidity"**
9. **Approve the transaction** in your wallet
10. **Copy the TX hash** from the confirmation or your wallet history

### Tips

- You need SOL for gas fees + rent (refundable when you close the position)
- Narrower range = higher fees but higher risk of going out of range
- Start with a small amount to test
- You can use "Ape In" for a faster single-token deposit

## Answer

Paste the TX hash / Solscan link on the site.
